# Assignment 5 - Comparative Binary Classification with XGBoost

## Objective
This project builds and compares binary classification models to predict whether a company is likely to go bankrupt.

## Dataset
Source: `training/data.csv` from the workshop repository.

Target variable: `Bankrupt?`

Positive class: `1` = bankrupt company

## Project files
- `Chinaza_Anulunko_assignment5.ipynb` - main notebook
- `summary.md` - short report summary
- `README.md` - project overview and run instructions

## Tools used
- pandas
- numpy
- scikit-learn
- xgboost
- matplotlib
- seaborn
